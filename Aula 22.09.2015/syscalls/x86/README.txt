Exemplo de chamada directa de um serviço de sistema (System call)
para versão de 32bits do Windows(Só funciona em versões até ao Windows XP, inclusivé).

Para gerar o executável, invoque o utilitário
"nmake" (versão microsoft do conhecido "make")
numa janela de consola do VS2010 ou VS2013 de desenvolvimento a 32bits.
